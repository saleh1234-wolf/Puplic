أداة  SpamCal لعمل مكالمات وهمية مزعجه لاي رقم بالعالم على termux :
   الأوامر :
$ pkg update

$ pkg upgrade

$ pkg install git -y

$ pkg install python -y

$ git clone https://github.com/gutclone/SpamCal

$ cd SpamCal

$ chmod 777 SpamCall.py

$ python SpamCall.py

بعد الانتهاء من تثبيت أداة وفتحها ضع رقم الهاتف الذي تريد الإتصال به مع رمز الدولة.
