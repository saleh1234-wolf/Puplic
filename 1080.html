اداة BruteConda:
هي أداة سريعة جدا في البحث عن لوحة المدير الخاصة بالموقع.
لغة البرمجة:
Python3
المنصة:
Linux
Termux
التحديث
apt update
apt upgrade
المتطلبات:
apt install git
apt install python3
التثبيت:
git clone https://github.com/AkDevilHunter/BruteConda.git
cd BruteConda
التفعيل:
chmod +x bruteconda.py
التشغيل:
python3 bruteconda.py
الإستخدام:
python3 bruteconda.py --url https://target.com --wordlist /path/to/wordlist
